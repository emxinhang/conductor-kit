# Tracks

This file is automatically maintained by Conductor. Manual edits may be overwritten.

## Track Status Legend
- `📅 Planned`: Đã có PRD/Spec, chờ lên Plan
- `[📅 Planned]`: Đã có Implementation Plan, chờ Code
- `[💻 Dev]`: Đang trong quá trình Code
- `[🧪 QA]`: Đoạn Check/Smoke Test
- `[✅ Completed]`: Đã hoàn thành và Release

## Transition Ledger
Mỗi track folder có thể chứa `CHANGELOG.md` ghi lại history chuyển status:
```
| Date | From | To | Agent | Note |
|------|------|----|-------|------|
```
Được ghi khi track chuyển status, thường qua `/conductor` hoặc `/update-knowledge` ở cuối session.
